Nuestro Proceso de Trabajo

Un flujo estructurado que garantiza resultados de alta calidad mientras mantiene una comunicación clara contigo en cada paso.

1. Kick-off y Referencias

¿Qué hacemos?

Analizamos tu documentación de diseño (lore, concept art, referencias) para entender completamente la visión de tu personaje. Si no tienes concept art, podemos trabajar con descripciones detalladas.

¿Qué necesitamos de ti?

  • Concept art o referencias visuales
  • Descripción detallada del personaje
  • Especificaciones técnicas (polycount, estilo, etc.)

Entregables

Documento de Especificación

Resumen detallado de los requisitos del personaje

Moodboard

Colección de referencias visuales aprobadas

Cronograma

Plan de trabajo con hitos y fechas clave

2. Blockout/Boceto 3D

¿Qué hacemos?

Creamos una versión simplificada del personaje usando formas básicas (cubos, esferas, cilindros) para establecer proporciones, silueta y volumen general. Esto permite ajustes rápidos antes de entrar en detalles.

¿Qué necesitamos de ti?

  • Feedback sobre proporciones y silueta
  • Aprobación para pasar a la siguiente fase

Entregables

Blockout 3D

Modelo simple en formato .fbx/.obj

Vistas 360°

Imágenes desde múltiples ángulos

Video Turntable

Rotación completa del modelo

3. Esculpido High-Poly

¿Qué hacemos?

Utilizando herramientas como ZBrush, añadimos todos los detalles complejos al modelo: arrugas, musculatura, pliegues de ropa, daños en armaduras, etc. Este modelo no está optimizado para juegos pero sirve como base para el baking de normales.

¿Qué necesitamos de ti?

  • Feedback sobre los detalles
  • Aprobación del diseño final
  • Confirmación de cualquier cambio de diseño

Entregables

High-Poly Model

Archivo ZTL (ZBrush) o FBX con subdivisiones

Renders Detallados

Close-ups de áreas con alto detalle

Approval Sheet

Documento para aprobación firmada

4. Retopología y UVs

¿Qué hacemos?

Creamos una nueva malla optimizada (low-poly) que sigue la silueta del high-poly pero con una topología limpia para animación. Luego desplegamos los UVs para preparar las texturas, maximizando el espacio disponible y minimizando distorsiones.

¿Qué necesitamos de ti?

  • Confirmación de polycount final
  • Feedback si se requieren ajustes

Entregables

Low-Poly Model

Modelo optimizado en .fbx/.obj

UV Layout

Imagen del mapeado UV

Reporte Técnico

Stats de triángulos, resolución UV, etc.

5. Texturizado PBR

¿Qué hacemos?

Utilizando Substance Painter, creamos texturas PBR (Albedo, Normal, Roughness, Metalness, etc.) que dan vida al modelo. Aplicamos materiales realistas o estilizados según tu requerimiento, incluyendo suciedad, desgaste y otros detalles que añadan autenticidad.

¿Qué necesitamos de ti?

  • Feedback sobre colores y materiales
  • Aprobación del estilo visual final

Entregables

Texturas PBR

Set completo en 4K/2K (PNG/TGA)

Material Preview

Vistas en diferentes condiciones de luz

Substance File

Archivo .spp para futuras ediciones

6. Rigging y Animación (si aplica)

¿Qué hacemos?

(Paquetes Estándar y Premium) Creamos un esqueleto (armature) profesional con controles fáciles de usar para animadores. Hacemos el skinning/pesado de vértices para que el modelo se deforme correctamente. Para el paquete Premium, añadimos un set de animaciones básicas listas para implementar.

¿Qué necesitamos de ti?

  • Especificaciones de rig (si son especiales)
  • Lista de animaciones requeridas (Premium)
  • Feedback sobre deformaciones

Entregables

Modelo Riggeado

FBX con controles de animación

Animaciones

Paquete Premium: 5-7 animaciones

Demo Reel

Video mostrando las animaciones

7. Entrega Final

¿Qué hacemos?

Empacamos todos los archivos finales en una estructura organizada, probamos los modelos en motores de juego (para paquete Premium) y preparamos documentación técnica. También ofrecemos una sesión de revisión final y soporte inicial para integración.

¿Qué necesitamos de ti?

  • Confirmación de recepción
  • Feedback final
  • Pago final (si aplica)

Entregables

Paquete Final

Todos los archivos organizados

Documentación

Guía de implementación

Soporte Inicial

15 días incluidos
